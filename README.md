# Illustrated Data Structures

Bienvenue sur le projet **Illustrated Data Structures**, une application interactive conçue pour visualiser et expérimenter avec diverses structures de données et algorithmes.

## Composants disponibles

### [Percolation](./app/percolation/README.md)
Simulation visuelle d'un système de percolation pour démontrer l'algorithme Union-Find. Permet de créer une grille $N \times N$, d'ouvrir interactivement des sites, et de verrouiller le système pour l'analyse.

---

## Démarrage rapide

Installez les dépendances et lancez le serveur de développement :

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur pour interagir avec les visualisations.

## Technologies utilisées
- [Next.js](https://nextjs.org/) (App Router)
- React
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
